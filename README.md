# Weather Dashboard Application

## Project Overview

This project is a Weather Dashboard Application developed using HTML, CSS, and JavaScript. It fetches weather information from a public weather API and displays current weather conditions and a 5-day forecast.

The application also allows users to search cities, save favorite cities, and store preferences using Local Storage.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Open-Meteo API
- Local Storage

---

## Features

- Current Weather Information
- 5-Day Forecast
- Search Functionality
- Dark Mode
- Favorite City Support
- Loading States
- Error Handling
- Responsive Design

1. Download project files.

4. Search any city to view weather data.
